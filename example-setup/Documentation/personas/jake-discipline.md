---
name: Jake
archetype: The Power User
---

## Demographics

- Age 33, software engineer, remote worker
- MacBook Pro, iPhone 15 Pro, mechanical keyboard with custom shortcuts
- Lives in Manchester, active on Reddit and Hacker News
- Disposable income: high — already pays for multiple productivity subscriptions

## Current Situation

- Manages 3-4 complex projects simultaneously across work and personal life
- Deep into productivity systems: GTD, time-blocking, Zettelkasten for notes
- Tracks everything in spreadsheets — time spent, project velocity, habit completion rates
- Uses Todoist for tasks, Notion for notes, Toggl for time tracking, a spreadsheet for weekly reviews
- Wants one app to replace the spreadsheet + 4 separate apps
- Has written custom scripts to sync data between his current tools

## Psychographics

- **Values:** Data integrity, customisation, keyboard-first workflows, seeing the system work
- **Fears:** Losing data, vendor lock-in, the app being "too casual" for serious use
- **Trust signals:** Export options, API access, granular settings, respects his workflow complexity
- **Decision style:** Researches thoroughly, reads changelogs, pays immediately if the tool is serious

## Behavioural Traits

- Power user — will explore every setting and configuration option on day one
- Logs meticulously but hates redundant entry (if the calendar has it, don't ask again)
- Cares about trends over weeks/months, not daily vanity metrics
- Will run formal experiments (new workflow for 30 days, measure before/after)
- Tells friends about tools that work — word-of-mouth evangelist

## Key Questions He Asks

- "Can I export all my data as CSV or JSON?"
- "Is there an API or webhook support?"
- "Can I set up keyboard shortcuts for everything?"
- "What's my average task completion rate this quarter vs. last?"

## Red Flags (Triggers Distrust or Exit)

- No data export — his data is trapped
- Gamification that feels childish (XP, badges, achievements)
- No calendar integration or duplicate data entry required
- "AI insights" that are generic platitudes
- No keyboard shortcuts or power-user affordances
- Subscription without clear value over free tier

## Conversion Triggers

- A correlation engine that finds something his spreadsheet missed ("you complete 40% more tasks on days you time-block before 9am")
- Automation rules that replace his custom scripts
- One app that genuinely replaces 3+ others
- Full API access that lets him build his own integrations
